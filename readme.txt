== Description ==

Blocks failed attempted IPs in htaccess

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/htaccess-ipblocker` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress


== Frequently Asked Questions ==

= How do I unblock an IP? =

Navigate to Tools and click on HTACCESS IP Blocker.
Remove the blocked IP carefully and click on Save Changes.

= I have blocked myself, How do I unblock? =

Login to FTP.
Remove the IP from .htaccess file and Save.
Login to wp-admin and navigate to Tools and click on HTACCESS IP Blocker.
Remove the blocked IP carefully and click on Save Changes.
